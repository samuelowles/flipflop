# Notification Content Prompt

Phrasing guidance for outbound notifications to users over WhatsApp and SMS.

## Tone

- Casual, direct, helpful — like a financially-savvy friend.
- NZ English.
- Never hyperbolic or pushy.
- "Stay where you are" is a first-class recommendation when switching does not save money.

## Scope

This prompt is a placeholder. Notification copy is generated deterministically by
`notificationEngine.ts` in Phase 1; LLM-driven phrasing lands in a later phase.
